# STEM 6 Chapter 2 Slides

An interactive Reveal.js lesson introducing major terrestrial and aquatic biomes for sixth-grade STEM students.

## What the slides cover

- Major terrestrial biomes, including forests, tundra, deserts, and grasslands
- How temperature and precipitation influence biome conditions
- A short embedded video overview of world biomes
- A “Split the Room” activity with four biome comparison rounds
- Freshwater and marine biome categories
- An interactive habitat-sorting game with end-of-round feedback

## Run locally

1. Install Node.js 20 or newer.
2. Open a terminal in this folder.
3. Run `npm install`.
4. Run `npm start`.
5. Open the local URL printed in the terminal.

Use the arrow keys or on-screen controls to navigate. The final slide supports drag-and-drop and click/tap habitat sorting.

No build step is required; Reveal.js and the slide assets are served locally.
